# Contribution Guidelines

Your contributions are welcome!

## Suggested Projects

To stay on the list, projects are suggested to be related to:

- Game
- Dataset
- Database
- Data Mining
- Artificial Intelligence
- Either:
  - free to use, or
  - open source (not limited to GitHub), or
  - has a documentated machine readable interface (e.g. database)

And must be either free to use, open source (not limited to GitHub), or have a documented machine-readable interface (e.g., database).

If unsure whether your project qualifies, open an issue to discuss.

## Formats

- Format: `- [project-name](https://example.com) - Description.`
- Preserve original project name capitalization (no need to convert letter case).
- Descriptions start with a capital letter and end with a period.
- Drop all A/An prefixes in descriptions.
- Descriptions should be in English.
- Sort links and categories alphabetically.
- New categories and improvements to existing ones are welcome.
